TypeScript - Introduction

A Superset of JavaScript for Strong Typing

What is TypeScript?

  • TypeScript is a programming language developed by Microsoft
  • It is an open-source superset of JavaScript
  • TypeScript adds static typing
  • Additional features like interfaces, classes, enums, and generics enhance code organization and maintainability.

Advantages of TypeScript

IDE Support

  • TypeScript is designed to integrate seamlessly with modern IDEs.
  • Popular editors like Visual Studio Code provide real-time error highlighting.
  • Developers can navigate code more efficiently and benefit from TypeScript's tooling.

Static Typing

  • TypeScript's static typing brings a new level of type safety to JavaScript, catching errors at compile time instead of runtime.
  • This reduces the risk of type-related bugs, making code more robust and reliable in the long run.
  • Early detection of type errors simplifies debugging and results in more stable applications.

Static Typing


              // JavaScript
              function greet(name) {
                return "Hello, " + name;
              }
              
              // TypeScript
              function greet(name: string): string {
                return "Hello, " + name;
              }

Type Inference

  • TypeScript can infer types based on the context, reducing the need for explicit annotations.
  • This makes code more concise and readable.

              /* TypeScript infers the variable type as number */
              let age = 25;
              
              /* TypeScript infers the return type as boolean */
              function isAdult(age: number) {
                return age >= 18;
              }

Interfaces

  • Interfaces define the structure of objects.
  • Enforce consistent shapes across your codebase.

            interface Person {
              name: string;
              age: number;
            }
            
            function displayPerson(person: Person) {
              console.log(person.name + " is " + person.age + " years old.");
            }

Enums

  • Enums define a set of named constants, giving more meaning to values.

            enum Color {
              Red,

              Blue,
            }
            
            let bgColor: Color = Color.Blue;

Abstract Classes

  • TypeScript allows defining abstract classes, which cannot be instantiated directly.
  • Abstract classes serve as a blueprint for other classes and may include abstract methods, which must be implemented in the derived classes. 

              // Abstract class 'Shape'
              abstract class Shape {
                abstract calculateArea(): number; // Abstract method, no implementation

                printArea(): void {
                  console.log('Area:', this.calculateArea());
                }
              }

              // Derived class 'Circle'
              class Circle extends Shape {
                constructor(private radius: number) {
                  super(); // Call to the constructor of the base class 'Shape'
                }

                calculateArea(): number {
                  return Math.PI * this.radius * this.radius;
                }
              }

              // Derived class 'Rectangle'
              class Rectangle extends Shape {
                constructor(private width: number, private height: number) {
                  super(); // Call to the constructor of the base class 'Shape'
                }

                calculateArea(): number {
                  return this.width * this.height;
                }
              }

              // Usage of classes
              const circle = new Circle(5);
              const rectangle = new Rectangle(4, 6);

              circle.printArea(); // Output: Area: 78.53981633974483
              rectangle.printArea(); // Output: Area: 24

Static Classes and Methods

  • TypeScript supports static classes and methods that belong to the class rather than instances.
  • Static methods are invoked directly on the class without creating instances.

              // Static class 'MathUtility'
              class MathUtility {
                static add(a: number, b: number): number {
                  return a + b;
                }

                static multiply(a: number, b: number): number {
                  return a * b;
                }
              }

              // Usage of static methods
              const sum = MathUtility.add(10, 20); // sum = 30
              const product = MathUtility.multiply(5, 8); // product = 40

Public, Protected, and Private Members

  • TypeScript supports access modifiers for class members, controlling their visibility and accessibility.

Public Members

  • Public members are accessible from anywhere, both within and outside the class.
  • By default, all class members are public if no access modifier is specified.

              class Car {
                public brand: string; // Public field
              
                constructor(brand: string) {
                  this.brand = brand;
                }
              
                public startEngine(): void { // Public method
                  console.log(this.brand + ' engine started.');
                }
              }
              
              const myCar = new Car('Toyota');
              console.log(myCar.brand); // Output: Toyota
              myCar.startEngine(); // Output: Toyota engine started.

Protected Members

  • Protected members are accessible within the class and its subclasses.
  • They cannot be accessed from outside the class hierarchy.

              class Vehicle {
                protected wheels: number; // Protected field
              
                constructor(wheels: number) {
                  this.wheels = wheels;
                }
              
                protected honk(): void { // Protected method
                  console.log('Honk!');
                }
              }
              
              class Car extends Vehicle {
                constructor() {
                  super(4);
                }
              
                public displayWheels(): void {
                  console.log('Number of wheels:', this.wheels); // Accessing protected member in a derived class
                }
              }
              
              const myCar = new Car();
              myCar.displayWheels(); // Output: Number of wheels: 4

Private Members

  • Private members are accessible only within the class where they are defined.
  • They are not accessible outside the class, including in its derived classes.

              class Person {
                private name: string; // Private field
              
                constructor(name: string) {
                  this.name = name;
                }
              
                private greet(): void { // Private method
                  console.log('Hello, my name is ' + this.name);
                }
              }
              
              const person = new Person('Alice');
              console.log(person.name); // Error: Property 'name' is private and only accessible within class 'Person'
              person.greet(); // Error: Property 'greet' is private and only accessible within class 'Person'

Generics

  • Generics allow writing flexible and reusable code.

            function reverse<T>(arr: T[]): T[] {
              return arr.reverse();
            }

            let numbers = [1, 2, 3, 4];
            let reversedNumbers = reverse(numbers); 
            /* Output: [4, 3, 2, 1] */
